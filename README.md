# Pytest API Automation Framework

## 📌 Overview

This project is a sample API automation framework built using pytest.

## 🚀 Features

* Pytest-based test execution
* Fixture-based architecture
* Parametrized test cases
* API testing using requests
* HTML reporting

## 📁 Project Structure

* tests/: test cases
* utils/: reusable API client
* config/: environment configs

## ▶️ How to Run

```bash
pip install -r requirements.txt
pytest --html=report.html --self-contained-html
```

## 🧪 Sample Tests

* API endpoint validation
* Status code verification
* Data validation

## 📊 Reporting

Generates HTML report after execution.
